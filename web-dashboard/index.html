API Integration Documentation

Use the following endpoints to integrate push notifications into your backend services.

1. Register Device & Map User

Register a device token and map it to a user ID. This is idempotent - safe to call multiple times.

POST /device

Request Body

{
  "userId": "henry930@gmail.com",
  "platform": "android",       // "android" or "ios"
  "token": "fcm_or_apns_token", // Device push token
  "deviceId": "unique_device_id" // e.g. UUID
}

Example (cURL)

curl -X POST /device \
  -H "Content-Type: application/json" \
  -d '{
    "userId": "henry930@gmail.com",
    "platform": "android",
    "token": "YOUR_FCM_TOKEN",
    "deviceId": "DEVICE_UUID"
  }'
Show React Native Client Example

Requires @react-native-firebase/messaging and react-native-device-info.

import messaging from '@react-native-firebase/messaging';
import DeviceInfo from 'react-native-device-info';

async function registerDevice(userId) {
  // 1. Get FCM Token
  const token = await messaging().getToken();
  
  // 2. Get Unique Device ID
  const deviceId = await DeviceInfo.getUniqueId();
  
  // 3. Register with backend
  await fetch('/device', {
    method: 'POST',
    body: JSON.stringify({
      userId: userId,
      token: token,
      deviceId: deviceId,
      platform: Platform.OS
    })
  });
}

2. Send Notification

Send a push notification to a specific user's registered devices.

POST /send

Request Body

{
  "userId": "henry930@gmail.com",
  "title": "Alert",
  "message": "Your transfer is complete"
}

Example (cURL)

curl -X POST /send \
  -H "Content-Type: application/json" \
  -d '{
    "userId": "henry930@gmail.com",
    "title": "Payment Received",
    "message": "You have received 0.5 BTC"
  }'
